# Insurance Management System

This is a Spring Boot REST API project for managing insurance policies.

## Features
- Create Policy
- Get All Policies
- Get Policy By ID
- Update Policy
- Delete Policy
- Global Exception Handling
- Validation

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## API Endpoints

POST /policies  
GET /policies  
GET /policies/{id}  
PUT /policies/{id}  
DELETE /policies/{id}
